# Сапёр и Морской бой

Статический проект для GitHub Pages. Работает без сервера, базы данных и внешних библиотек.

## Что внутри

- `index.html` — разметка приложения.
- `styles.css` — Apple-like минималистичный интерфейс, адаптивность, темы и анимации.
- `app.js` — логика Сапёра, Морского боя, ИИ, кампании, рекордов и localStorage.
- `manifest.webmanifest`, `sw.js`, `icons/` — PWA, офлайн-кэш и иконки.

## Функции

- Сапёр: 4 сложности, безопасный первый ход, флаги, таймер, рекорды, ежедневное испытание.
- Морской бой: ручная и автоматическая расстановка, поворот кораблей, 4 сложности ИИ, добивание кораблей, ежедневный бой.
- Профиль: XP, звания, статистика, винрейт, точность, экспорт JSON.
- Кампания: Новичок → Матрос → Капитан → Адмирал, открытие визуальных стилей.
- Темы: системная, белая, тёмная. Дополнительные стили открываются за XP.
- Звуки: можно включать/выключать.

## Публикация на GitHub Pages

1. Распакуй архив.
2. Загрузи содержимое папки `apple_games_hub_full` в корень репозитория.
3. Открой Settings → Pages.
4. Выбери Deploy from branch → `main` → `/root`.
5. Сохрани и дождись публикации.

Если после обновления на GitHub Pages отображается старая версия, очисти кэш сайта или открой страницу в режиме инкогнито. Service Worker кэширует файлы для офлайн-режима.
