# Сапёр и Морской бой

Статический проект для GitHub Pages. Никаких серверов и внешних библиотек.

## Как опубликовать

1. Загрузи содержимое папки в корень репозитория.
2. Включи GitHub Pages: Settings → Pages → Deploy from branch → main / root.
3. Открой опубликованную ссылку.

## Файлы

- `index.html` — разметка сайта.
- `styles.css` — Apple-like минималистичный интерфейс, темы и адаптивность.
- `app.js` — логика Сапёра и Морского боя.
- `manifest.webmanifest` и `icons/` — базовая PWA-информация и иконка.
